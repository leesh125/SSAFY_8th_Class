●Comparable, Comparator
  - 객체 비교 : 크다, 작다, 같다
    -> 순서를 결정

  ○ Comparable
    - CompareTo(Obj) , return type : int
      : 순서가 결정되어야 할 대상객체들이 스스로 비교 판단 가능하도록 하게 함

      - 결과값 
        음수 : 내가 작아요. (교환 안함)
          0   : 둘이 같다. (교환 안함)
        양수 : 내가 커요(교환이 일어남)
    
※ 바뀌지 않는 것은 코드에 받아두고 바뀌는 것은 빼내야 한다 -> 재사용성 증가
