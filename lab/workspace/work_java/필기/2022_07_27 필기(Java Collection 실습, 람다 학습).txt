  ○ List 유형 : 순서 有, 원소 중복 허용
  ○ Set 유형 : 집합, 순서 無, 원소 중복 허용 X
  ○ Map 유형 : Key-value(원소), 순서 無, 원소 중복 허용
                    Key는 유일값으로 관리

● 람다식
  - 식별자 없이 실행 가능한 함수(익명함수)
  - 함수형 프로그래밍언어 지원
  - Java : Java8 지원 -> 멀티 패러다임 언어 성격 지님
    -> 익명 inner 객체(1개의 메소드)

  ○ 람다식 표현
    - (매개변수, ...) -> {수행문장}
    -> 1. 매개변수가 1개이면 괄호 생략
        2. 수행 문장 1문장이면 블록 생략
        3. 수행 문장 1문장이면서, 문장이 return 문이면 return 키워드 생략

  ○ 작성 순서
   1. 메소드 名 지우기
   2. 매개변수 와 블록({}) 사이에 -> 넣고
   3. 타입 추론 가능하면 생략
