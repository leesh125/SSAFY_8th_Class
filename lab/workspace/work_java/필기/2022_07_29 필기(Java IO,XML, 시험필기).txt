● XML
  - extensible markup language

  ○ well-formed 규칙
    1. 시작태그 - 종료태그 한쌍
    2. 속성은 시작태그에만 올 수 있고 속성 = "값" 형태
    3. Markup 포함관계는 가능, overlap 불가
    4. Root element는 단 1개(반드시 존재)
    5. 주석(<!-- -->)은 XML 선언부 이후에 등장
    6. <, & 파싱에 위배 -> 엔티티 사용대체, CDATA Section
