● 자바
.java(소스 파일) -> 컴파일 -> .class(바이트 코드) -> jvm이 해석하여 운영체제에 독립적

프로세스 - 구동중인 프로그램
쓰레드 - 프로세스 안에서 독립적으로 실행

개발환경 - jdk
실행환경 - jre = jvm + api

● 자바 데이터 타입
  ○ 원시 타입(Primitive Type) :  원시값을 데이터(값)로
    
  ○ 참조 타입(Reference Type or Non-Primitive Type) : 참조값 -> 데이터(값)
    - 참조값: 메모리 주소 X

● 변수
  ○ 멤버변수
    - 클래스 변수(static)
    - 인스턴스 변수(non-static)
  ○ 지역 변수
    - 참조값 -> 주소(address) -> 실제 값
