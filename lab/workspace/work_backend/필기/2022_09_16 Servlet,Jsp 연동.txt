● JSP : Java Server Page
- 응답 출력을 손쉽게 만들 수 있도록 지원하는 동적 웹페이지 기술

● JSP 구성요소
1. Directive Tag : <%@ 지시어 속성="값" %> (지시어: page, include ,taglib)
2. Declaration Tag : <%! 멤버선언 %>
3. Scriptlet : <% 자바 실행문 %> -> _jspService() 안으로 복사
4. Expression Tag : <%= 표현식 %> => out.출력메소드(표현식)
5. Comment : <%--  --%>
6. JSP: action Tag
7. JSTL
8. EL

● MVC 디자인패턴
- App 구성요소들의 역할을 나누어 설계
  ○ Model : B/L, DAL, Domain Object(Value Object, Data Transfer Object)

  ○ View : P/L(UI) => Input / output

  ○ Controller : C/L


● Page 이동기법
1. Redirection 기법 
: 서버가 브라우저에게 새롭게 이동할 페이지 정보를 응답으로 보내고
 이걸 받은 브라우저가 다시 새 페이지를 요청해서 응답받는 방식

2. Forwarding 기법
: 서버가 서버내의 다른 페이지로 이동시키고 다른 페이지가 처리한 결과가
  응답 되어지는 방식
