● JSP include 방식
- 정적 페이지 include 방식 : file 내용 copy & paste
  -> <%@ include file=" " %>

- 동적 페이지 include 방식 : 해당 file을 실행시키고 자신의 page를 계속해서 실행하는 방식
  -> <jsp:include page=" " />

● EL(Expression Language)
- JSP 안의 구성요소로 출력 목적, JSTL 함께 사용하기 위한 목적 표현언어(표기법)

  - 연산자 지원 : +, *, empty, not, eq
  - 내장객체지원 : 11개(10개: map, 1개: Beans(getter, setter)- pagecContext
  - null 처리 : 출력 - 빈문자열
                 산술연산 : 0으로 처리
                 참조 : 참조 X -> NullpointerEx X

  ○ 사용하는 위치
    - output 출력하는위치 : 출력역할
    - JSTL 속성값 위치 : 값 전달 역할

  ○ Syntax
    - ${first:second}
    - EL 내장객체 이름, 4개본관함에 저장 attribute "key"



● URL 관리
  - 상대경로 : 현재 경로 기준으로 상대적 경로
  - 절대경로 : 온전한 경로 -> Ex. http:ip:port/contextPath/path/file
  - 반절대(상대)경로 : 절대 경로 성격 + 상대 경로 성격
